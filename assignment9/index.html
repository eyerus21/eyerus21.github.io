Q4: Define a function sum() and a function multiply() that sums and multiplies (respectively) all the numbers in an array of numbers. For example, sum([1,2,3,4]) should return 10, and multiply([1,2,3,4]) should return 24.
Q4:a) Sum of elements in array:

Ans:- function sum takes an array as its rguments and returns the sum of the elements

    const sum=(arr)=>{
        return arr.reduce((sum,element)=>sum+element,0);
    }

Q4:b. Product of elements in array

Ans:- function product takes array as its arg and returns product of all the elements

    const multiply=(arr)=>{    
        return arr.reduce((mult,element)=>mult*element,1);;
    }

Q5: Define a function reverse() that computes the reversal of a string. For example, reverse("jag testar") should return the string "ratset gaj".

Ans:- function reverse takes string as its arg and returns reversed form it

        const reverse=(str)=>{
            return str.split('').reverse().join('');
        }

Q7: Write a function filterLongWords() that takes an array of words and an integer i and returns the array of words that are longer than i.

Ans:- function filterLongWords takes array and key as its args and returns filtered array

    const filterLongWords = (words,key) =>{
      const filteredArray = words.filter((elt, i, arr) =>{
          return elt.length > key;
        });
        return filteredArray;
    };
