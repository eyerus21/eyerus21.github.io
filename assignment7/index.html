Q1: Define a function max() that takes two numbers as arguments and returns the largest of them. Use the if-then-else construct available in Javascript.

Ans:- function max returns the maximum of 2 arguments 

  function max(a, b) {
    if (a > b) {
      return a;
    } else {
      return b;
    };
  }

Q2: Define a function maxOfThree() that takes three numbers as arguments and returns the largest of them.

Ans:- function maxOfThree takes 3 numbers as arguments and returns the largest 

  function maxOfThree(a, b, c) {
    return max(max(a, b), c);
  
  }

Q4:a) Sum of elements in array:

Ans:- function sum takes an array as its rguments and returns the sum of the elements

  const sum = (arr) => {
    return arr.reduce((summation, elem) => summation + elem,0);
  };

Q4:b. Product of elements in array

Ans:- function product takes array as its arg and returns product of all the elements

  const product = (arr) => {
    return arr.reduce((product, elem) => product * elem,1);
  };

Q5: Define a function reverse() that computes the reversal of a string. For example, reverse("jag testar") should return the string "ratset gaj".

Ans:- function reverse takes string as its arg and returns reversed form it

  const reverse = text => {
      let reversed='';
      let size = text.length;

      for(let i=size-1; i>=0; i--){
        reversed += text[i];
      }
      return reversed;
  };

Q6: Write a function findLongestWord() that takes an array of words and returns the length of the longest one.

Ans:- function findLongestWord accepts array of strings and returns length of the longest.

    const findLongestWord = arrayOfWords => {
      if(arrayOfWords==null || arrayOfWords.length===0 || arrayOfWords==undefined)
        return 0;

      let lengthOflongestWord = arrayOfWords[0].length;
      arrayOfWords.forEach(element => {
        if(element.length > lengthOflongestWord){
          lengthOflongestWord = element.length;
        }
      });
      return lengthOflongestWord;
    };

Q7: Write a function filterLongWords() that takes an array of words and an integer i and returns the array of words that are longer than i.

Ans:- function filterLongWords takes array and key as its args and returns filtered array

    const filterLongWords = (words,key) =>{
      const filteredArray = words.filter((elt, i, arr) =>{
          return elt.length > key;
        });
        return filteredArray;
    };
